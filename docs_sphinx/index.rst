Description
-----------
This page aims to gather tools/ function for solving signal processing problems like denoising, spectral analysis, filtering and convulution, feature detection and so on. 
It aims to document the `GitHub repository <https://github.com/neby68/Signal_processing_toolbox>`_ of the same name.
This page may be referenced by the `Electrical and neural signal processing examples' documentation <https://neby68.github.io/electrical_and_neural_signal_processing/>`_.
